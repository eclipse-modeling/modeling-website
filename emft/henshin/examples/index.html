Henshin - Examples

This page lists documented examples of Henshin transformations. The type column indicates whether the transformation is endogenous (operating on a single model) or exogenous (translating one or more source models to one or more target models). The other columns indicate which editor was used for the example, how the transformation is executed, and whether it contains an application of any analysis methods.
