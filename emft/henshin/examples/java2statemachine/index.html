Henshin Example: Java2StateMachine

contributed by Stefan Jurack, Johannes Tietje and Christian Krause

This transformation is an exogenous one solving the Reengineering case of the Transformation Tool Contest (TTC) 2011. This example primarily shows the use of transformation unit and parameter passing. Some of the details could be modeled differently and more efficiently using nested rules.

A detailed description of the transformation can be found here:

  • Stefan Jurack, Johannes Tietje: Solving the TTC 2011 Reengineering Case with Henshin. Proceedings of TTC'11, Electronic Proceedings in Theoretical Computer Science (EPTCS), 2011, Volume 74.
