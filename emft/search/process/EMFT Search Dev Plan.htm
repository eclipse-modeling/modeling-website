EMF Model Search Dev Plan

Last Modification (August 7th 2007) by Lucas Bigeardel & Jacques Lescot

You are warmly invited to comment on and/or join the project development plan discussions. Please send all feedbacks to the Modeling EMFT newsgroup & corresponding bugzillas.

Terminology

Task Priorities
[P1] Critical
[P2] Major
[P3] Normal
[P4] Low
[P5] Lowest

Releases Milestones

EMF Search is basically a "+2w" Eclipse project, mainly having dependencies on EMF and, potententially "+3 weeks", considering dependencies on MDT UML2, UML2 tools & OCL (UML2 Search is an example and thus should be delivered with 1 week delay ... to be discussed).

According to this timeline, we can define some milestones tags as :

[Mn] : Eclipse 3.4 Integration Milestone "n"
[Mn + p] : Eclipse 3.4 Integration Milestone "n" + "p" weeks
[MRCn + p] : Eclipse 3.4 Release Candidate Milestone "n" + "p" weeks

Tasks Breakdown (Architectural)

Core
  • (50%)[MS_CORE_100][P2][M1+2w] : Find dialog for loaded model (search engine based)
  • [MS_CORE_200][P3][M1+2w] : Study Teneo & search engine exploration/loading
  • [MS_CORE_300][P4][M7+2w](Tentative) : Design Pattern Search
UI
  • [MS_UI_RESULTS_100][P1][M4+2w] : SelectionToViewer interface definition
  • [MS_UI_RESULTS_200][P2][M4+2w] : Open Graphical Editor Toggle Mode
  • (75%) [MS_UI_RESULTS_300][P2][M4+2w] : Hierarchical Organization
  • [MS_UI_RESULTS_400][P4][M6+2w] : Rational Common approach particpants organization (UML2 Only ?)
  • [MS_UI_PARTICIPANTS_100][P3][M5+2w] : Hierarchical Organization
  • [MS_UI_PARTICIPANTS_200][P3][M5+2w] : Participants Hierarchy Filter
Generation
  • (75%) [MS_CODEGEN_100][P2][MRC3+2w] : Jet Template base generation
  • (75%) [MS_CODEGEN_200][P2][MRC3+2w] : Genmodel integration (Ask for an ext point to EMF)
  • (50%) [MS_CODEGEN_300][P2][M3+2w] : Ecore Search generation
  • (50%) [MS_CODEGEN_400][P3+][M4+2w] : UML2 Search generation
Examples
  • (75%) [MS_EXAMPLES_100][P2][M4+2w] : Add UML2 search as an example
Documentation
  • (50%) [MS_DOCUMENTATION_100][P4][M1+2w] : API Documentation + Process + Roadmap
Releng
  • (50%) [MS_RELENG_100][P2][M2+2w] : Runtime
  • [MS_RELENG_200][P3][M2+2w] : SDK
  • [MS_RELENG_300][P3][M6+2w] : Doc
  • [MS_RELENG_400][P4][M4+2w] : Examples
Testing
  • [MS_TESTS_100][P2][M5+2w] : Add EMF Search Engine Core Tests
  • [MS_TESTS_100][P2][M5+2w] : Add EMF Search Ecore Tests
  • [MS_TESTS_100][P2][M6+2w] : Add EMF Search UML2 Tests Tests
