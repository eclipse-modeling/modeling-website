Installing

The b3 aggregator can be installed with p2 from these repositories:
(these URLs are intended for use with p2 - they can not be viewed in the browser)

  • For Eclipse 3.6 IDE: http://download.eclipse.org/modleing/emft/b3/updates-3.6 (Recommended)
  • For Headless 3.6 operation: http://download.eclipse.org/modeling/emft/b3/headless-3.6

Archived sites

If you want a local copy of an archived update site:

  • http://download.eclipse.org/modeling/emft/b3/archivedsites-3.6/.

The b3 engine

The b3 engine and DSL editor is currently only available from SVN. See instructions how to set up development environment on the b3 wiki.
