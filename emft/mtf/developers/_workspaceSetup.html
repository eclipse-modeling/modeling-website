Here's how to set up your workspace for developing MTF

  1. Download and install any Eclipse distribution - download
  2. Install Teneo and Hibernate as described on this wiki page: http://wiki.eclipse.org/Teneo/Hibernate/Download_and_Install
  3. Download and import the MTF Team Project set file into your workspace.
    • extssh (for Eclipse committers)
    • pserver (for anonymous access)
