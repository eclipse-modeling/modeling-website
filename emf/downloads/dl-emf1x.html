Requirements
  • Use Eclipse Version 2.1.2 for EMF 1.1.1/EMF 1.1.1.1, Eclipse Version 2.1.1 for EMF 1.1.0, or Eclipse Version 2.1 for EMF 1.0.2.
  • For XSD 1.x runtimes, you must use a version 1.3.1 Java 2 Standard Edition SDK only.
  • XSD Runtime (required only if you plan to define or use XML Schema based models)


  • Important! The version, build date, and build level for the XSD Runtime that you use must match that of the EMF Runtime that you use. For example, you could use the XSD Runtime (xsd_1.1.0_20030620_1105VL.zip) with the EMF Runtime (emf_1.1.0_20030620_1105VL.zip).
