Eclipse Modeling Framework - Plan Summary

Project Name: Eclipse Modeling Framework (EMF)

Project Leader(s): Ed Merks and Elena Litani

Project Description: EMF is a modeling framework and code generation facility for building tools and other applications based on a structured data model. EMF includes the XML Schema Infoset Model (XSD) project and an EMF-based implementation of Service Data Objects (SDO). See http://eclipse.org/emf.

Project Components:
EMF and SDO components and XSD components.

Executive Summary:

The EMF project is currently working on the 2.2 release. For details see EMF 2.2 Project Plan.
