Project Name: M2T

Project Leaders: Paul Elder

Project Description: The Model to Text (M2T) project focuses on the generation of textual artifacts from models. See http://www.eclipse.org/m2t.

Executive Summary: The objectives of this project are: to provide implementations of industry standard and defacto Eclipse standard model-to-text engines; to provide exemplary development tools for these languages; and to provide common infrastructure for this languages.
