The 1.x release(s) of UML2 were based on an iterim draft of the UMLTM 2.0 specification (040502) that was produced by the original Superstructure FTF (Finalization Task Force).

The 2.0.x release(s) are based on the (soon to be) finalized version 2.1(.1) of the specification. The 2.0 release introduced a number of major semantic and API changes - its focus was on conformance with the standard and reduction of memory footprint. Please see the project plan(s) for details.
