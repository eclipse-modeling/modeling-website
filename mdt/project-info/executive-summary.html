Project Name: MDT

Project Leaders: Kenn Hussey

Project Description: The Model Development Tools (MDT) project focuses on big "M" modeling within the Modeling project. See http://www.eclipse.org/mdt.

Executive Summary: The objectives of this project are to provide an implementation of industry standard metamodels and exemplary tools for developing models based on those metamodels.
